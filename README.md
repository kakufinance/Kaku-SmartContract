## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Deploy

```shell
$ forge script script/KAKUScript.s.sol:KAKUScript --rpc-url <your_rpc_url> --private-key <your_private_key>
```
